🟢 Server 24 Jam Nonstop

Bot Minecraft berbasis Player yang dirancang untuk menjaga server Aternos tetap aktif selama 24 jam nonstop dengan cara otomatis masuk ke dalam server. Sangat cocok digunakan untuk server SMP, server komunitas, maupun server pribadi agar tidak mati ketika tidak ada pemain yang online.

---

✨ Fitur Utama

- ✅ Auto Connect
  Bot akan otomatis bergabung ke server saat server aktif.

- ♻️ Auto Reconnect
  Secara otomatis mencoba terhubung kembali apabila koneksi terputus atau server melakukan restart.

- 🛡️ Menjaga Server Tetap Aktif
  Membantu mencegah server Aternos berhenti karena tidak ada pemain yang online.

- 🌐 Siap Deploy di Render
  Sudah dilengkapi fitur Self Ping sehingga dapat berjalan 24/7 secara gratis menggunakan Render.

- 🔐 Dukungan Plugin Login
  Kompatibel dengan server Paper, Spigot, dan Bukkit, termasuk fitur login otomatis (Auto Auth) untuk plugin autentikasi.

- 📜 Log Aktivitas
  Semua aktivitas chat dan status bot akan ditampilkan langsung pada console.

---

🛠️ Persyaratan

Sebelum memulai, pastikan Anda telah memiliki:

- Akun GitHub
- Server Aternos
- Akun Render (untuk menjalankan bot selama 24 jam)
- Node.js (jika dijalankan secara lokal)
- Koneksi internet yang stabil

⚙️ Cara Penggunaan

1. Nyalakan server Aternos Anda.
2. Bot akan otomatis mencoba masuk ke dalam server.
3. Jika server restart atau koneksi terputus, bot akan melakukan reconnect secara otomatis.
4. Apabila menggunakan Render, Anda dapat membuka URL aplikasi untuk melihat status bot.
5. Semua aktivitas bot dapat dipantau melalui log pada console.
